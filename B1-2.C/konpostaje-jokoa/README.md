# Konpostaje Jokoa

## Deskribapena
Konpostaje Jokoa web aplikazio interaktiboa da, ikasleek konpostaje prozesua simulatzen laguntzen duena. Joko honek erabiltzaileei hezetasuna, tenperatura, pH, C/N erlazioa eta denbora bezalako faktoreak kontrolatzeko aukera ematen die, konpostaje prozesua nola funtzionatzen duen ikasteko.

## Proiektua Abiarazteko

1. **Fitxategiak Kargatu**: Proiektua abiarazteko, `public` direktorioan dauden fitxategiak kargatu behar dituzu web zerbitzari batera. Hona hemen gomendatutako zerbitzu batzuk:
   - [GitHub Pages](https://pages.github.com/)
   - [Netlify](https://www.netlify.com/)
   - [Vercel](https://vercel.com/)

2. **Proiektua Argitaratu**:
   - **GitHub Pages**: Proiektua GitHub-en biltegira bultzatu eta GitHub Pages gaitzeko aukera aktibatu. Hautatu `public` direktorioa iturri gisa.
   - **Netlify edo Vercel**: `public` direktorioa drag-and-drop bidez igo dezakezu edo zure GitHub biltegia konektatu.

3. **URL-a Partekatzea**: Proiektua argitaratu ondoren, URL-a partekatu ikasleekin, euren nabigatzaileetan jokoan sartzeko.

4. **Sarbidea Probatu**: Jokoak funtzionatzen duela ziurtatu behar da nabigatzaile desberdinetan, erabiltzaile guztientzat esperientzia atsegina eskaintzeko.

## Fitxategiak

- `public/index.html`: Jokoaren interfazearen egitura eta estiloak lotzen dituen HTML dokumentua.
- `public/style.css`: Jokoaren estiloak definitzen dituen CSS fitxategia.
- `public/script.js`: Jokoaren logika eta interaktibitatea ezartzen dituen JavaScript fitxategia.
- `src/js/main.js`: Jokoaren funtzionalitatea hobetzeko edo kodea modulotan antolatzeko erabil daitekeen JavaScript fitxategia.
- `src/css/main.css`: `public/style.css` fitxategian definitutako estiloak osatzen edo gainditzen dituen CSS fitxategia.
- `test/gameplay.test.js`: Jokoaren funtzionalitatea probatzeko test kasuak.
- `package.json`: npm konfigurazio fitxategia, mendekotasunak, scriptak eta proiektuaren metadata zerrendatzen dituena.
- `.gitignore`: Git-ek ignoratu beharreko fitxategiak eta direktorioak zehazten dituena.
